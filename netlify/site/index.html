OPR Card Generator — Relay

This is the backend relay for the OPR Card Generator. It is not the app — the app is hosted separately on GitHub Pages.

Endpoints:

  • GET /army?id=<id> — Army Forge list enriched with core rule descriptions.
  • GET /health — health check.

Free, non-monetized community use only — the condition under which OnePageRules permits relaying this data.
